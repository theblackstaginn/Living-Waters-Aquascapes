Process

Simple, direct, and builder-led—so the finished feature fits the space naturally.

  1. Consultation

    We walk the space and define goals: sound, flow, access, maintenance, and budget.

  2. Design Direction

    We set the stone character and feature layout so it looks intentional and natural.

  3. Build & Install

    Stone placement, plumbing, and flow tuning—built for longevity and stability.

  4. Walkthrough

    Final adjustments and care guidance so it stays beautiful over time.
